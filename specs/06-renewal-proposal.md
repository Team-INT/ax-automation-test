## 1. 프로젝트 개요

### 1.1 배경

재단법인 굿네이버스 글로벌 임팩트(이하 GNGIF)는 **26개국에서 621개 협동조합, 13개 사회적기업, 5개 소액금융기관**을 운영하는 사회적경제 생태계 구축 기관입니다. 그러나 현재 웹사이트(gngif.org)는 이 규모와 임팩트를 효과적으로 전달하지 못하고 있습니다.

### 1.2 리뉴얼 목적

| 목적 | 상세 |
|------|------|
| **파트너십 확대** | 글로벌 파트너 기관 및 기업 후원사 유치를 위한 전문적 브랜드 이미지 구축 |
| **후원 동선 확보** | 기업 파트너십 + 개인 후원 이중 전환 경로 마련 |
| **임팩트 가시화** | 사업 성과를 데이터와 스토리로 증명하는 콘텐츠 체계 구축 |
| **글로벌 신뢰** | 핵심 페이지 영문화로 해외 파트너 대응력 강화 |

### 1.3 대상 사이트

| 항목 | 내용 |
|------|------|
| URL | https://www.gngif.org |
| 현재 CMS | WordPress (커스텀 테마 `gngif`) |
| 다국어 | 한국어/영문 (비대칭 운영 중) |
| 분석 도구 | Google Analytics (G-SNQJF25FCS) + WebTrends |

### 1.4 타겟 오디언스

| 순위 | 대상 | 사이트 역할 |
|------|------|-----------|
| **1순위** | 글로벌 파트너 기관 + 기업 후원사 | 파트너십 확대, 기업후원 유치 (사이트 핵심 목적) |
| **2순위** | 일반 후원자 (개인 기부) | 개인후원 전환 동선 확보 |
| **3순위** | 내부 직원 + 채용 대상자 | 인재 확보, 조직 브랜딩 |

> 사이트 톤: **기관/기업 대상의 신뢰감 있는 톤** 중심, 개인후원 동선도 확보

## 2. 현황 분석 (AS-IS)

### 2.1 현재 사이트 구조

```
메인 (/) ─┬─ 재단소개 (소개/미션/투명경영/조직도)
          ├─ 사업소개 (사업지역/협동조합/소액금융/사회적기업/소셜비즈니스)
          ├─ 소식 (스토리/공지사항/뉴스/채용)
          └─ 후원 (기업후원)
```

- 한국어 18페이지, 영문 9페이지 (비대칭)
- GNB 4개 메뉴

### 2.2 핵심 문제

| # | 문제 | 증거 | 비즈니스 영향 |
|---|------|------|-------------|
| 1 | **전환 동선 부재** | "기업후원" 단일 페이지만 존재. 개인후원/해외파트너 경로 없음 | 파트너십·후원 유치 기회 손실 |
| 2 | **임팩트 비가시** | 메인에 성과 수치 없음. 텍스트 슬로건만 | CSR 담당자가 보고 자료를 만들 수 없어 이탈 |
| 3 | **개인후원 동선 부재** | 기업후원만 존재. 개인 기부 경로 전무 | 재원 다각화 실패 |

### 2.3 기술적 문제

- WordPress 커스텀 테마 유지보수 부담
- jQuery 의존, 성능 최적화 미비
- 한글 slug(`/사업-지역-new`), 구조화 데이터 미적용
- 모바일 반응형 기본 수준

## 5. 정보 구조 (IA)

### 5.1 GNB 설계

**데스크톱**
```
[로고]  재단소개  사업소개  임팩트  소식  후원·파트너십  채용  [KO|EN]
```

**6개 메뉴** (기존 4개에서 확장)
- `임팩트`: 성과 데이터 전용 섹션 신설 → 1순위 타겟 공통 요구
- `후원·파트너십`: 기업/개인/해외 3분화 → 1·2순위 타겟 수용
- `채용`: 공지사항에서 독립 → 3순위 타겟 전용 메뉴

**모바일**
```
┌─────────────────────────────────┐
│ [로고]              [햄버거]     │
│─────────────────────────────────│
│         (페이지 콘텐츠)          │
│─────────────────────────────────│
│ [파트너십 문의]    [후원하기]     │  ← Sticky Bottom
└─────────────────────────────────┘
```

## 6. 디자인 컨셉 (3종)

3가지 디자인 방향을 제안합니다. 각 컨셉은 HTML 시안으로 제작되어 있으며, 브라우저에서 직접 확인 가능합니다.

### 6.1 컨셉 A: DATA STORYTELLING — 데이터 스토리텔링형

> "성과로 증명하는 조직"

임팩트 데이터를 시각적 스토리로 풀어내는 **인터랙티브 연례보고서** 스타일. 스크롤하며 차트가 빌드되고, 수치가 살아 움직이는 경험.

| 항목 | 내용 |
|------|------|
| **컬러** | Deep Forest `#1B4332` + Emerald `#2D6A4F` + Gold `#D4A847` + Cream `#FEFCF3` |
| **타이포** | Plus Jakarta Sans (모던) + Noto Sans KR |
| **레이아웃** | 스크롤 드리븐 내러티브, 벤토 그리드, 수평 스크롤 섹션 |
| **인터랙션** | SVG 차트 빌드, 카운터 틱업, 패럴랙스, clip-path 이미지 리빌 |
| **핵심 타겟** | 기관/기업 파트너 (1순위 타겟 최적) |

**주요 섹션 구성**
1. 다크 오버레이 히어로 + 패럴랙스
2. 스크롤 프로그레스바
3. 임팩트 넘버스 (카운터 애니메이션 + 미니 바차트)
4. SVG 차트 섹션 (연도별 성장, 지역별 분포)
5. 수평 스크롤 사업영역 카드
6. 글로벌 맵 (SVG 세계지도 + 펄싱 도트)
7. 수혜자 스토리 카드 (clip-path 리빌)
8. 함께하기 CTA + 푸터

**레퍼런스**: charity: water, ONE Campaign, Kiva

### 6.2 컨셉 B: MAGAZINE EDITORIAL — 매거진 에디토리얼형

> "사람을 잇는 따뜻한 이야기"

따뜻한 매거진 스타일로 수혜자와 참여자의 실제 이야기를 전달. 사진과 텍스트의 조화로 **감성적 연결**을 만드는 에디토리얼 접근.

### 6.3 컨셉 C: SCROLL NARRATIVE — 스크롤 내러티브형

> "한 사람의 변화가 시작되는 곳"

월드비전 캠페인에서 영감을 받은 강력한 스크롤 스토리텔링. 한 수혜자의 이야기를 **시네마틱하게** 풀어내며 감성적 몰입을 극대화.

| 항목 | 내용 |

| **컬러** | Dark Charcoal `#1a1a2e` + Green `#2D6A4F` + Gold `#D4A847` + Cream `#FEFCF3` |
| **타이포** | Noto Serif KR (감성 헤드라인) + Noto Sans KR (본문) |
| **레이아웃** | 풀스크린 섹션, 스크롤 시 순차 텍스트 등장, 시네마틱 구성 |
| **인터랙션** | IntersectionObserver 텍스트 리빌, 다크→라이트 전환, 반복 CTA |
| **핵심 타겟** | 캠페인/모금 대상 (후원 전환 최적) |

**주요 섹션 구성**
1. 다크 시네마틱 히어로 (캠페인 헤드라인 + CTA)
2. 수혜자 스토리 (스크롤 시 텍스트 순차 등장)
3. 드라마틱 통계 ("지금 이 순간에도...")
4. 수혜자 카드 그리드 (에티오피아/캄보디아/르완다)
5. 사업분야 카드
6. FAQ 아코디언
7. 반복 CTA (후원/파트너십) + 푸터

**레퍼런스**: 월드비전 1000명의 소녀들, charity: water, UNHCR

## 9. 콘텐츠 계획

### 9.1 유지 콘텐츠
- **4대 핵심사업** 소개 체계 (협동조합, 소액금융, 사회적기업, 소셜비즈니스) — 사업 정체성

### 9.2 업데이트 필요
- 사업국 수, 수혜자 수, 협동조합 수 등 **수치 데이터** → 고객사 제공 대기
- 인재채용 정보 갱신

### 9.3 보강 필요
- **미션/비전 페이지** — 조직 철학과 전략 방향 명확화

### 9.4 신규 콘텐츠 (수급 필요)

| 콘텐츠 | 용도 | 수급 방법 |
|--------|------|----------|
| 수혜자 스토리 (3~5건) | 스토리 캐러셀, 임팩트 스토리 | 현장 인터뷰/촬영 또는 기존 자료 활용 |
| 기업 파트너십 사례 (3~5건) | Case Study 페이지 | 기존 파트너사 협의 |
| 현장 사진/영상 | 히어로, 사업 소개 | 기존 아카이브 또는 신규 촬영 |
| 파트너사 로고 | 파트너 섹션 | 고객사 제공 |
| 임팩트 데이터 | 대시보드, 카운터 | 고객사 제공 대기 |

> 임팩트 수치는 현재 사이트 기준 수치를 placeholder로 기재하고, 최종 데이터 확정 시 교체
