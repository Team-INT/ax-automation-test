# 고객사 요구사항 정리 — gngif.org 리뉴얼

> 수신일: 2026-02-12
> 상태: IA 제안서 v1.0 피드백으로 수신

---

## 1. 타겟 오디언스 우선순위

| 순위 | 대상 | 비고 |
|------|------|------|
| **1순위** | 글로벌 파트너 기관 + 기업 후원사 | 사이트 핵심 목적: 파트너십 확대, 기업후원 유치 |
| **2순위** | 일반 후원자 (개인 기부) | 개인후원 동선 반드시 확보 |
| **3순위** | 내부 직원 + 채용 대상자 | — |

- 사이트 톤: **기관/기업 대상 신뢰감** 중심, 개인후원 동선도 확보
- 해외 사업장 중심 자립 지원 사업체 특성 반영

---

## 2. 콘텐츠 유지/변경

| 구분 | 내용 |
|------|------|
| **반드시 유지** | 4대 핵심사업 소개 체계 (협동조합, 소액금융, 사회적기업, 소셜비즈니스) — 사업 정체성 |
| **업데이트** | 사업국 수, 수혜자 수, 협동조합 수 등 수치 데이터 → 최신 기준 교체 예정. 인재채용 갱신 |
| **보강** | 미션/비전 페이지 — 조직 철학과 전략 방향 명확화 |
| **신규 추가** | 개인후원 페이지 (필수). 임팩트 성과 대시보드 (검토 중) |

---

## 3. 임팩트 데이터

- 최신 데이터는 고객사 측 제공 예정
- 제안서에는 **현재 사이트 기준 수치 기재**, 최종 데이터는 추후 교체
- 데이터 확정 시 즉시 공유 예정

---

## 4. 영문 사이트 범위

| 영문 제공 | 한국어만 |
|-----------|---------|
| 재단 소개 (About) | 뉴스/공지사항 |
| 4대 사업 소개 (Programs) | 채용 |
| 임팩트/성과 (Impact) | 기타 자주 업데이트 콘텐츠 |
| 파트너십/후원 안내 (Partnership) | — |

- 영문은 **정적 콘텐츠 위주**, 자주 업데이트되는 콘텐츠는 한국어만
- 글로벌 파트너가 주요 타겟이므로 **영문 품질 중요**

---

## 5. 자체 브랜드 비중 (더네이버스커피, 메리쿱)

- 소셜비즈니스 사업의 **성과 사례**로서 의미 있음
- 해당 섹션(소셜비즈니스) 내에서 충분히 소개
- 사이트 전체에서 **과도한 비중 지양**
- 메인 페이지: 사업 성과 중 하나로 **간결하게 노출**
- 상세 소개: 소셜비즈니스 하위 페이지에서 처리

## IA 반영 사항 요약

| 기존 제안 (v1.0) | 변경 방향 (v1.1) |
|-------------------|------------------|
| P1(기업)+P2(투자자) 1순위 | 글로벌 파트너+기업후원 통합 1순위 |
| P6(소비자) 브랜드 중심 | 개인후원자로 재정의 (2순위) |
| `/brands` GNB 독립 메뉴 | 삭제 → 소셜비즈니스 하위 유지 |
| 개인후원 동선 없음 | 개인후원 페이지 필수 추가 |
| 영문 전 페이지 1:1 | 핵심 4개 페이지만 영문 |
| 미션/비전 현행 유지 | 보강 필요 (철학·전략 명확화) |
| 톤: 따뜻하고 인간적 | 기관/기업 신뢰감 + 개인후원 동선 확보 |
