## 컨셉 A: DATA STORYTELLING — 데이터 스토리텔링형

> 📁 `concepts/concept-a-data-storytelling.html`

### 방향성
임팩트 데이터를 시각적 스토리로 풀어내는 인터랙티브 연례보고서 스타일. 스크롤하며 차트가 빌드되고, 수치가 살아 움직이는 경험.

### 키워드
`데이터 드리븐` `인터랙티브` `역동적` `성과 중심` `스크롤 스토리텔링`

### 디자인 특성

| 항목 | 내용 |
|------|------|
| **컬러** | Deep Forest #1B4332 + Emerald #2D6A4F + Gold #D4A847 + Cream #FEFCF3 |
| **타이포** | Plus Jakarta Sans (모던, 약간 라운드) + Noto Sans KR |
| **레이아웃** | 스크롤 드리븐 내러티브, 벤토 그리드, 수평 스크롤 섹션 |
| **인터랙션** | SVG 차트 빌드, 카운터 틱업, 패럴랙스, 프로그레스바, clip-path 이미지 리빌 |
| **이미지** | Unsplash 실사 — 커뮤니티, 현장, 데이터 시각화 배경 |

### 주요 섹션
1. 다크 오버레이 히어로 + 패럴랙스
2. 스크롤 프로그레스바
3. 임팩트 넘버스 (카운터 애니메이션 + 미니 바차트)
4. SVG 차트 섹션 (연도별 성장, 지역별 분포)
5. 수평 스크롤 사업영역 카드
6. 글로벌 맵 (SVG 세계지도 + 펄싱 도트)
7. 스토리 카드 (clip-path 리빌)
8. CTA + 푸터

### 적합 상황
- 임팩트 성과를 강조하고 싶을 때
- 데이터 중심 의사결정자(기업/기관 파트너) 타겟
- 연례보고서/제안서와 연계 가능

### 레퍼런스
- charity: water (스크롤 데이터 스토리텔링)
- ONE Campaign (인터랙티브 임팩트)
- Kiva (데이터 시각화)

## 컨셉 B: MAGAZINE EDITORIAL — 매거진 에디토리얼형

> 📁 `concepts/concept-b-magazine-editorial.html`

### 방향성
따뜻한 매거진 스타일로 수혜자와 참여자의 실제 이야기를 전달. 사진과 텍스트의 조화로 감성적 연결을 만드는 에디토리얼 접근.

### 키워드
`따뜻함` `스토리텔링` `매거진` `사람 중심` `에디토리얼` `커뮤니티`

### 디자인 특성

| 항목 | 내용 |
|------|------|
| **컬러** | Warm Cream #FBF7F0 + Soft Olive #6B7F5E + Clay #C4856A + Dark Brown #2C1810 |
| **타이포** | Noto Serif KR (헤드라인, 에디토리얼 따뜻함) + Noto Sans KR (본문) |
| **레이아웃** | 매거진 스프레드, 메이슨리 그리드, 비대칭이지만 균형 잡힌 구성 |
| **인터랙션** | 이미지 패럴랙스, 카드 호버 리프트, 타이프라이터 인용문, CSS 캐러셀 |
| **이미지** | Unsplash 실사 — 웃는 얼굴, 일하는 손, 커뮤니티 모습, 웜톤 |

### 주요 섹션
1. 스플릿 히어로 (대형 사진 + 텍스트)
2. 피처드 스토리 (매거진 스프레드)
3. 풀쿼트 섹션 (다크 배경, 페이드인)
4. 메이슨리 스토리 그리드 (6개 아이템, 스태거드 등장)
5. 에디토리얼 사업영역 카드 (사진 배경 + 호버 텍스트 리빌)
6. 테스티모니얼 자동 캐러셀
7. "함께하기" CTA
8. 푸터

### 적합 상황
- 감성적 연결과 공감을 이끌어내고 싶을 때
- 개인 후원자/스토리 공감형 방문자 타겟
- 브랜드의 인간적 면모를 강조하고 싶을 때

### 레퍼런스
- Acumen (매거진 스토리텔링)
- 아름다운재단 (따뜻한 비영리 톤)
- Patagonia Action Works (에디토리얼)

## 컨셉 C: SCROLL NARRATIVE — 스크롤 내러티브형

> 📁 `concepts/concept-c-scroll-narrative.html`

### 방향성
월드비전 1000명의 소녀들 캠페인에서 영감을 받은 강력한 스크롤 스토리텔링. 한 수혜자의 이야기를 시네마틱하게 풀어내며 감성적 몰입을 극대화.

### 키워드
`시네마틱` `감성 내러티브` `몰입형` `스크롤 드리븐` `캠페인` `행동 유도`

### 디자인 특성

| 항목 | 내용 |
|------|------|
| **컬러** | Dark Charcoal #1a1a2e + Green #2D6A4F + Gold #D4A847 + Cream #FEFCF3 |
| **타이포** | Noto Serif KR (감성 헤드라인) + Noto Sans KR (본문) |
| **레이아웃** | 풀스크린 섹션, 스크롤 시 순차 텍스트 등장, 시네마틱 구성 |
| **인터랙션** | IntersectionObserver 텍스트 리빌, 다크→라이트 전환, 반복 CTA, 아코디언 FAQ |
| **이미지** | Unsplash 실사 — 시네마틱 인물 사진, 현장 풍경, 감성적 클로즈업 |

### 주요 섹션
1. 다크 시네마틱 히어로 (캠페인 헤드라인 + CTA)
2. 수혜자 스토리 (스크롤 시 텍스트 순차 등장 — 르완다 커피 농부 엠마뉘엘)
3. 드라마틱 통계 ("지금 이 순간에도...")
4. 수혜자 카드 그리드 (에티오피아/캄보디아/르완다)
5. 사업분야 카드 (5개)
6. FAQ 아코디언
7. 반복 CTA (후원/파트너십)
8. 푸터

### 적합 상황
- 캠페인/모금 중심 페이지
- 감성적 몰입으로 행동(후원/파트너십)을 유도하고 싶을 때
- 특정 이슈/프로젝트를 부각할 때

### 레퍼런스
- 월드비전 1000명의 소녀들 (worldvision.or.kr/campaign/25-1000girls)
- charity: water (캠페인 스토리텔링)
- UNHCR 캠페인 페이지
