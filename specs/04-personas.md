## 우선순위 매트릭스 (고객사 확정)

> v1.1: 고객사 피드백 반영. 글로벌파트너+기업후원 1순위, 개인후원 2순위, 채용 3순위로 조정.

```
비즈니스 임팩트 (높음)
     │
     │   [1순위: 핵심 타겟 — 파트너십·기업후원]
     │
     │   P1. 기업 CSR/ESG 담당자       P7. 해외 파트너 기관
     │   (기업후원 유치 핵심)             (파트너십 확대 핵심)
     │
     │   [2순위: 개인후원]
     │   개인 후원자 (재원 다각화)
     │
     ├──────────────────────────────────────────── 사용 빈도 (높음)
     │
     │   [3순위: 채용·내부]
     │   P4. 채용 지원자               P5. 언론/미디어
     │   (인재 확보)                    (간접 영향력)
     │
     │   [지원]
     │   P3. 국제개발 종사자            P6. 브랜드 소비자
     │   (학술적 신뢰 강화)              (소셜비즈니스 성과 사례)
     │
```

| 순위 | 페르소나 | 사이트에서 원하는 것 | 현재 최대 병목 |
|------|---------|-------------------|--------------|
| **1** | 기업 CSR/ESG 담당자 | 파트너십 사례, 임팩트 데이터, 사업소개서 PDF | 기업 전용 동선 없음 |
| **1** | 해외 파트너 기관 | 영문 사업 소개, 파트너십 프로세스, 실적 데이터 | 영문 콘텐츠 부실 |
| **2** | 개인 후원자 | 후원 방법, 임팩트 시각화, 간편 결제 | 개인후원 페이지 자체가 없음 |
| **3** | 채용 지원자 | JD, 조직 문화, 지원 프로세스 | 공지사항에 채용 묻힘 |
| **-** | 언론/미디어 | 팩트시트, 미디어킷, 홍보담당 연락처 | 프레스센터 없음 |
| **-** | 국제개발 연구자 | 인용 가능한 성과 데이터, 발간물 | 데이터가 홍보 수준 |
| **-** | 브랜드 소비자 | 카페 위치, 제품 구매, 브랜드 스토리 | 소셜비즈니스 하위에서 접근성 개선 필요 |

## 핵심 여정 병목 3선

### 병목 1: 기업 파트너십 전환 단절

```
CSR 담당자 검색 유입 → 메인 → "후원" 클릭 → 기업후원 단일 페이지
                                                    ↓
                                               일반 문의 폼 (기업명 필드 없음)
                                                    ↓
                                               응답 대기 (자동확인 메일 없음)
                                                    ↓
                                              ❌ 이탈 or 경쟁 기관으로 전환
```

**개선**: 파트너십 전용 랜딩 → 유형별(후원/투자/마케팅) 분기 → 전용 문의폼(기업명/예산/관심분야) → 자동확인메일+사업소개서PDF 발송 → Calendly 미팅 예약

### 병목 2: 개인후원 전환 부재

```
일반인 검색/SNS 유입 → gngif.org 메인
                              ↓
                         "후원" 메뉴 클릭 → 기업후원 페이지만 존재
                              ↓
                         개인이 참여할 수 있는 경로가 없음
                              ↓
                        ❌ 이탈 (개인 후원 의사가 있어도 전환 불가)
```

**개선**: 후원·파트너십 섹션에 개인후원 독립 페이지 → 후원 유형(정기/일시/사업지정) 안내 → 임팩트 시각화("당신의 후원이 만드는 변화") → 간편결제(카카오페이/네이버페이) → 후원 후 감사 + 성과 피드백

> 참고: 자체 브랜드(더네이버스커피, 메리쿱)는 소셜비즈니스 하위에서 충분히 소개하되, 사이트 전체 비중은 적정 수준 유지 (고객사 방침)

### 병목 3: 임팩트 데이터 신뢰 부족

```
투자자/연구자/기자 유입 → 사업소개 페이지 탐색
                              ↓
                         홍보성 텍스트만 있음 (출처/방법론 없음)
                              ↓
                         연간보고서 PDF 찾기 어려움 or 미최신
                              ↓
                        ❌ "신뢰할 수 있는 데이터가 없다" → 이탈
```

**개선**: 임팩트 대시보드(연도/국가/사업별 필터) + 연간보고서 아카이브 + 임팩트 측정 방법론 공개 + 학술/연구 데이터 인용
