# 개선안 레퍼런스

> 원문 라인 번호 기준. 설정 충돌 여부 검증 완료.

## A. 한이서 등장 타이밍 — 편의적 우연 제거

### A-1. 2화 복선 삽입 (신규)

**파일**: `02_오만한_헌터의_목줄을_쥐다.md`  
**위치**: 395행 직후 (`존나 편리한데?` 다음, `그나저나 문밖을 나서면` 이전)  
**삽입 이유**: 마력 감지가 막 활성화된 직후이므로, 장벽 밖의 기척을 처음 감지하는 것이 자연스러움.

**삽입문:**

```
그런데.

감지가 열리는 순간, 장벽 바깥에서 아주 짧게 ── 바늘 끝만 한 시선이 스쳤다.

감시인지, 관측인지.
확신할 수는 없다. 하지만 의도가 담긴 기척이었다.

누군가 바깥에서 안쪽을 들여다보고 있었다.
```

**설정 검증**: 마력 감지 활성화(392행)가 직전에 발생. 코어를 갓 흡수한 상태이므로 감지 정밀도가 낮아 "확신할 수 없다"는 서술이 자연스러움. 한이서가 A급 랭커이므로 기척을 완벽히 숨기되, 백사언이 "의도의 잔향"만 감지하는 수준이 설정상 합당함.

### A-2. 3화 한이서 대사 교체

**파일**: `03_시스템_에러.md`  
**위치**: 175행 (한이서 대사 원문 1문장)

**원문:**

```
"클리어 이후에도 마력이 불안정해서 월별로 모니터링하고 있어요. 그래서 오늘 우연히 현장 근처에 있었는데 흥미로운 일이 터졌더라고요."
```

**교체문:**

```
"솔직하게 말할게요. 오늘 우연히 온 건 아니에요."

백미러 너머로 그녀의 시선이 잠시 흔들렸다.

"며칠 전부터 이 게이트 주변에서 비정상적인 마력 잔향이 잡혔거든요. 클리어된 게이트에서 새로운 파장이 발생하는 건 이론적으로 불가능해요. 그래서 직접 확인하러 왔는데."

핸들을 가볍게 두드렸다.
```

**그 이후 원문 177행~179행(`한이서가 부드럽게 코너를 꺾었다` ~ `A급 스킬의 마력 신호가~`)은 그대로 유지.**

**설정 검증**: 한이서는 레길론 게이트 공략 멤버(원문 확인)이므로 해당 게이트의 마력 잔향을 추적할 동기와 역량이 모두 있음. "우연"을 삭제하고 "이상 징후 추적"으로 전환해도 이후 대사("A급 스킬 마력 신호의 소멸과 재점화")와 자연스럽게 연결됨. 추가로 "한이서가 백사언 이전부터 이 현상에 관심을 갖고 있었다"는 상위 떡밥 라인이 열림.

## B. 협회 안전망 블러핑 — 법무팀 역량 보정

**파일**: `07_전장은_게이트_밖에서도.md`  
**위치**: 148~151행

**원문:**

```
"원본 데이터는 협회 안전망에 자동 전송되도록 세팅해 뒀지요."

이거는 블러핑이다.
놈들이 지금 당장 확인할 방법은 없다.
```

**교체문:**

```
"원본 데이터는 외부 정보상의 클라우드에 분산 백업되어 있습니다. 제가 48시간 이상 헌터 앱에 생체 로그인을 하지 않으면, 자동으로 협회 공익 제보 채널과 주요 언론사 세 곳에 동시 전송되도록 세팅해 뒀거든요."

절반은 블러핑이다.

클라우드 백업은 사실이다. 브로커가 해놨으니까.
하지만 '48시간 자동 전송'은 지어낸 거다.

관건은 이 새끼들이 지금 당장 진위를 확인할 수 없다는 점.

정보상의 클라우드는 신원 추적이 불가능한 다크 서버다.
협회 내부망으로 조회해봐야 어디에 뭐가 심어져 있는지 알 도리가 없다.
확인하려면 정보상을 직접 찾아야 하는데, 그건 며칠이 걸리는 일이다.

임재훈 변호사가 볼펜 끝으로 메모패드를 톡톡 두들겼다.

"그 정보상이라는 쪽의 신원이 확인되지 않는 이상, 유효한 증거 보전으로 인정하기는——"

"변호사님."

내가 말을 잘랐다.

"지금 중요한 건 법정에서 인정되느냐가 아니잖습니까. 내일 아침 뉴스 화면에 차도연 헌터의 육성이 깔리면, 소송 결과가 나오기 전에 주가부터 떨어지죠."

임재훈의 볼펜이 멈췄다.
```

**그 이후 153행(`"설령 녹음이 존재한다 해도."`)부터는 원문 그대로 이어짐.**

**설정 검증**: 5화에서 브로커가 `클라우드에 백업 해두었습니다`라고 회신한 원문 확인. 클라우드 백업=사실, 48시간 자동 전송=허구의 이중 구조. 법무팀이 한 번 반격("증거 보전 인정 불가")한 뒤 백사언이 본질("여론 리스크")로 꺾는 구조이므로, 법무팀이 바보가 아니라 "리스크 계산에서 밀린 것"으로 읽힘. 8화에서 `신화의 시가총액 몇천 억이 증발`이라는 주인공 독백과도 논리적으로 일관.

## C. 톤 단조로움 개선 — 7개 씬

> 핵심 원칙: "감정이 없는 것"이 아니라 "감정을 억누르는 데 숙달된 것"으로 전환.  
> 기존의 냉소적 어미와 마무리는 전부 유지. 그 **직전**에 1~3문장의 날것 감정을 삽입한 뒤 즉시 차단하는 구조.

---

### C-1. 1화: 손가락이 짓밟히는 순간

**파일**: `01_히든_특성_절대_몰수가_발동됩니다_.md`  
**위치**: 73행 (`"…!"`) 직후, 75행 (`짓눌린 손가락 위로`) 직전

**원문 흐름**: `"…!"` → 바로 차도연 묘사

**삽입문:**

```
이가 맞물렸다.

반사적으로 치밀어 오르는 것을 ── 삼켰다.

이 바닥에서 감정은 사치고, 사치에는 비용이 붙는다.
```

**검증**: 이후 등장하는 `습관처럼 고개를 들지 않았다`(63행, 이전 대사)의 톤과 일관. "삼켰다"라는 동사가 10년간의 학습된 억압을 함축. 직후의 차도연 관찰 모드로 자연스럽게 복귀.

---

### C-2. 2화: 데스나이트 격파 직후 착지

**파일**: `02_오만한_헌터의_목줄을_쥐다.md`  
**위치**: 312행 (`이 지옥 같은 게이트 안에서, 처음으로 두 다리로 서 있었다.`) 직후, 314행 (`……나쁘지 않군.`) 직전

**삽입문:**

```
손끝이 떨렸다.

분노도, 희열도 아니었다.
이름을 붙일 수 없는 무언가가 흉골 깊은 곳에서 한 번 크게 울리고 사라졌다.
```

**검증**: "두 다리로 서 있었다"라는 서술이 10년간 기어 다녔던 과거와 대비되는 전환점. 여기에 신체 반응(떨림)을 넣되 감정을 명명하지 않음으로써 캐릭터의 자기 인식 한계를 보여줌. `나쁘지 않군`이 허세가 아니라 감정 억압의 결과물로 재해석됨.

---

### C-3. 3화: 한이서 차 안에서 과거를 말하는 장면

**파일**: `03_시스템_에러.md`  
**위치**: 161행 (`어차피 숨길 정보가 아니다.`) 교체

**원문:**

```
담담하게 말했다.

어차피 숨길 정보가 아니다.
```

**교체문:**

```
담담하게 말했다.
목소리가 담담한 건 연습의 결과다.
이 대사를 몇 번이나 했는지 모른다.
관공서 서류 칸에, 고용주의 면접장에서, 채굴판 막사의 가슴팍에.
반복하면 무엇이든 닳는다. 감정도 마찬가지다.
```

**검증**: 1화에서 10년간 불법 채굴판과 노동 계약을 전전한 이력이 확인됨. 관공서·고용주·막사라는 구체적 장소가 이 이력과 일치. 4화의 고아원 원장 회상(`원장님은 드물게 선한 어른이었다`)과도 톤이 연결됨. 이후 한이서의 반응(`……그랬군요.`)이 더 무게를 갖게 됨.

### C-4. 4화: 피를 토하고 쓰러지기 직전

**파일**: `04_내_허락_없이_타오르지_마.md`  
**위치**: 101행 (`한이서의 얼굴이 사색이 되었다.`) 직후, 103행 (`"저기요! 무슨 일입니까!"`) 직전

**삽입문:**

```
시야가 흐려졌다.
이 감각을 알고 있다.
채굴판에서 탈수로 쓰러지기 직전의 그 느낌.
또 바닥으로 돌아가는 건가.
```

**검증**: 채굴판 탈수 경험은 1화의 `무릎 감각은 진작에 뒤져 버렸다` 등 열악한 노동 환경 묘사와 일관. "또 바닥으로 돌아가는 건가"는 주인공의 근원적 공포(무력한 상태로의 추락)를 암시하며, 이후 전 화차에 걸친 공격적 행보("다시는 바닥에 가지 않겠다")의 동기 부여 역할. 직후 `끌려가는 건 안됩니다`(119행)와 감정적으로 직결됨.

### C-5. 5화: 파티원의 경외를 받는 장면

**파일**: `05_헌터로는_처음이라.md`  
**위치**: 365행 (`은혜라니.`) 교체 확장

**원문:**

```
은혜라니.
입막음용 합의금치고는 반응이 거창하네.
```

**교체문:**

```
은혜라니.

채굴판에서 마석 포대를 끌고 지나가면, 헌터들은 시선을 주지 않았다.
주더라도 벽지나 장판을 보는 눈이었다.
그게 10년이다.

지금 이 사람들은 나를 보고 있다.

……입막음용 합의금치고는 반응이 거창하네.
```

**검증**: 1화의 `어디를 가든 부모도, 출신도, 학력도 없는 나의 목숨값은 가벼웠다`와 일관. "벽지나 장판을 보는 눈"이라는 비유가 짐꾼 시절의 투명인간 취급을 구체화. 마지막 줄은 원문의 냉소적 마무리를 그대로 유지하되, 앞에 깔린 과거 대비 덕분에 냉소가 "방어기제"로 읽히게 됨. 본인이 직접 "기뻤다"고 말하지 않으므로 캐릭터 톤 유지.

### C-6. 6화: 최하얀과 처음 만나는 장면

**파일**: `06_물음표.md`  
**위치**: 79행 (`혼자 벅차올라서 떠들고 있네.`) 직후, 81행 (`그러나 꽂히는 단어가 있었다.`) 직전

**삽입문:**

```
묘한 기시감이 스쳤다.
고아원의 원장님이 아이들 앞에서 그림책을 읽을 때 저런 눈을 했었다.
좋아하는 것을 말할 때 눈이 빛나는 사람.

── 쓸데없는 연상이다.
```

**검증**: 4화에 `원장님은 드물게 선한 어른이었다`는 회상 존재. 같은 인물을 다른 맥락에서 호출하므로 설정 충돌 없음. "쓸데없는 연상이다"로 즉시 차단하여 캐릭터 톤 유지. 최하얀이 단순한 도구("사냥개")가 아니라 주인공의 내면에 균열을 낼 수 있는 존재로 가능성을 열어둠. 이후 81행 `그러나 꽂히는 단어가 있었다`에서 분석 모드로 자연 복귀.

---

### C-7. 9화: 류진혁 마검기 몰수 후 스킬 충돌

**파일**: `09_두번째_능력은.md`  
**위치**: 326~330행 교체

**원문:**

```
"진짜 적응 안 되네…… "

이를 악물고 억지로 두 힘을 짓눌러 통제했다.

절대 동화를 얻을 때 겪은 통증에 비해선 아무것도 아니니까.
```

**교체문:**

```
"진짜 적응 안 되네……"

이를 악물고 억지로 두 힘을 짓눌러 통제했다.

절대 동화를 얻을 때 겪은 통증에 비하면 아무것도 아니다.
그런데 이상한 건 통증이 아니었다.

류진혁의 코어가 동화되는 순간, 찰나처럼 스친 감각.
칼자루를 쥔 손의 감촉. 누군가를 벤 횟수. 피 냄새에 대한 무감각.
스킬뿐 아니라, 이 기술을 써온 놈의 잔상까지 섞여 들어온 건가.

불쾌했다.
```

**그 이후 원문 332행 (`다행히도 이번엔 금방 통증이 사그라들었다.`)은 그대로 유지.**

**검증**: 류진혁이 검(칼)을 사용하는 것은 원문 전체에서 확인(`검집`, `칼자루를 집어 들었다`, `검은 코트와 검집` 등). "누군가를 벤 횟수"는 감찰팀장으로서의 실전 경력과 일관. 이 삽입은 두 가지 기능을 수행:

1. **단기**: 두 번째 몰수의 감정적 질감을 첫 번째(차도연)와 차별화. 차도연 몰수 때는 순수한 역전 쾌감이었다면, 류진혁 몰수에서는 "남의 것이 침투하는 불쾌함"이라는 새로운 감각 추가.
2. **장기**: 몰수를 반복할수록 타인의 잔재가 누적된다는 복선. 향후 정체성 흔들림, 또는 흡수한 코어들의 간섭이라는 갈등 라인으로 확장 가능. 프롤로그의 `시스템이 저 여자가 위험하다고 경고합니다`처럼 시스템이 주인공을 제어하려는 떡밥과도 연결 여지 있음.

## 적용 시 주의사항

- 모든 삽입문은 **앞뒤 원문을 수정하지 않고 사이에 끼워넣는 구조**로 설계됨. 삭제가 필요한 곳은 원문을 명시적으로 `교체`라고 표기함.
- 삽입 분량은 개당 2~5문장. 전체 회차 분량에 미치는 영향 미미.
- C-1~C-7은 독립적이므로 순서 무관하게 개별 적용 가능. 단, A-1(2화 복선)과 A-2(3화 대사 교체)는 반드시 세트로 적용해야 복선-회수 구조가 성립.
