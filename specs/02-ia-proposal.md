## 0. 왜 리뉴얼해야 하는가

현재 gngif.org는 **26개국, 621개 협동조합, 13개 사회적기업**이라는 상당한 규모의 사업을 운영하고 있지만, 사이트가 이 임팩트를 전달하지 못하고 있다.

### 핵심 문제 3가지

| 문제 | 증거 | 비즈니스 영향 |
|------|------|-------------|
| **1. 전환 동선 부재** | 후원은 "기업후원" 단일 페이지. 투자자/개인 후원/해외 파트너 경로 없음 | 파트너십·투자 유치 기회 손실 |
| **2. 임팩트 비가시** | 메인 히어로에 성과 수치 없음. 홍보성 텍스트만. Kiva는 3대 지표를 애니메이션으로 표시 | 기업 CSR 담당자가 보고 자료를 만들 수 없어 이탈 |
| **3. 개인후원 동선 부재** | 기업후원만 존재. 개인 기부 경로가 전무. 사이트가 B2B에만 맞춰져 있어 일반 후원자 전환 불가 | 개인 후원 수익 기회 상실, 재원 다각화 실패 |

### 벤치마크 대비 현재 위치

Kiva, Acumen, Warby Parker, Patagonia 등 유사 기관 대비 임팩트 데이터 시각화, CTA 전략, 대상별 동선 분리에서 **모두 최하위 수준**. (상세: `03-benchmark.md`)

## 3. GNB 설계

### 데스크톱

```
[로고]  재단소개  사업소개  임팩트  소식  후원·파트너십  채용  [KO|EN]
```

**6개 메뉴 (기존 4개에서 확장)**

확장 이유: 타겟별 진입점이 부족했던 문제를 해결. `후원·파트너십`은 기업/개인/해외 3분화로 1·2순위 타겟 모두 수용. `채용`은 3순위 타겟 전용 독립 메뉴. 자체 브랜드(더네이버스커피, 메리쿱)는 사업소개 > 소셜비즈니스 하위에서 충분히 소개.

```
┌─────────────────────────────────┐
│ [로고]              [햄버거]     │
│─────────────────────────────────│

│                                 │
```

## 6. 후속 논의 사항 (우선순위순)

### 즉시 결정 필요

| # | 항목 | 선택지 | 결정 영향 |
|---|------|--------|----------|
| 1 | **기술 스택** | WordPress 리빌드 / Next.js + Headless CMS / Next.js + 정적 | 전체 개발 일정과 운영 방식 결정 |
| 2 | **임팩트 데이터 소스** | 수동 업데이트 / DB 연동 / API | 대시보드 설계 범위 결정 |
| 3 | **후원 결제 시스템** | 자체 결제 / 카카오페이·네이버페이 연동 / PG사 직접 연동 | 개인후원 페이지 구현 범위 결정 |
